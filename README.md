# Jogodavelha

Simples jogo da velha em terminal, foi meu primeiro projeto em programação.

"                           COMO JOGAR                                   "
INPUTS --> 1 1
Será marcada a posição x = 1, e y = 1

INPUTS --> 2 3
Será marcada a posição x = 2, e y = 3

A partir daí, é só jogar 0//
